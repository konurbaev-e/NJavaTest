# Q10
Какие из приведенных выражений будут истинными при выполнении кода. Возможно несколько вариантов ответа.

```java
String a = new String("hello");
String b = new String(a);
String c = a;
char[] d = { 'h', 'e', 'l', 'l', 'o' };
```

    1) (a == "hello")
    2) (a == b)
    3) (a == c)
    4) a.equals(b)
    5) a.equals(d)

## Правильный ответ
3) (a == c)

4) a.equals(b)
